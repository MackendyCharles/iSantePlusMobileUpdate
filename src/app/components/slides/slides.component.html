Bienvenue Sur iSantePlus Mobile

Isanteplus mobile est une version mobile d'IsantePlus - système de dossiers médicaux Développé à partir de OpenMRS en Haïti.




Pourquoi iSantePlus Mobile

Isanteplus mobile couvrira la plupart des fonctionnalités de l'application Web isanteplus, y compris l'enregistrement des patients, la prise de notes de visite, la capture des signes vitaux, l’enregistrement des vaccins, etc.






CHARESS

Cette version sera évoluée dans le but de satisfaire les besoins des prestataires de soin.
